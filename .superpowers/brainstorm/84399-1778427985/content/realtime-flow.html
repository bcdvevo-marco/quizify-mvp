📡 WebSocket Olay Zinciri

PLAYER_JOINED
nickname, team_id → Lobiye yayın
🚪
Lobi aşaması
🚀
GAME_STARTED
Host başlatır → tüm oyuncular oyun ekranına geçer
QUESTION_START
question_data, time_limit, start_timestamp → Geri sayım başlar
❓
Her soru için tekrar
💬
ANSWER_COUNT
Kaç kişi cevapladı (isim yok) → Host ekranında bar doluyor
QUESTION_END
Süre doldu veya herkes cevapladı → Cevap göster, puan hesapla
⏱️
🏆
LEADERBOARD_UPDATE
Güncel sıralama → Soru arası skor tablosu
GAME_END
Son soru bitti → Final ekranı, sonuçlar DB'ye kaydedilir

🗺️ Sayfa Yapısı (Next.js Rotaları)

Host Rotaları

/ → Landing
/giris → Giriş / Kayıt
/dashboard → Quiz Listesi
/quiz/yeni → Yeni Quiz
/quiz/[id]/duzenle → Editör
/oyun/[id]/lobi → Lobi Yönetimi
/oyun/[id]/kontrol → Oyun Kontrolü
/oyun/[id]/sonuclar → Analiz & Rapor

Oyuncu Rotaları

/katil → PIN / Link Girişi
/katil/[pin] → Takmaad & Takım
/oyna/[session] → Bekleme Lobisi
/oyna/[session]/soru → Soru + Cevap
/oyna/[session]/skor → Soru Arası Skor
/oyna/[session]/bitis → Kişisel Sonuç
