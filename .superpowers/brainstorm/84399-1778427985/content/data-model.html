Veri Modeli — Veritabanı Tabloları

Supabase PostgreSQL · Tablolar arası ilişkiler gösterildi

👤 users

  • 🔑 id
  • 📧 email
  • 👤 name
  • 🖼️ avatar_url
  • 🕐 created_at

Host hesapları. Supabase Auth ile yönetilir.

📋 quizzes

  • 🔑 id
  • 🔗 host_id → users
  • 📝 title
  • 🖼️ cover_image_url
  • 📌 status (draft/published)
  • 🕐 created_at

❓ questions

  • 🔑 id
  • 🔗 quiz_id → quizzes
  • 📝 text
  • 🖼️ image_url
  • ⏱️ time_limit (sn)
  • 🔢 position

✅ options

  • 🔑 id
  • 🔗 question_id → questions
  • 📝 text
  • ✓ is_correct
  • 🔢 position (A/B/C/D)

🎮 game_sessions

  • 🔑 id
  • 🔗 quiz_id → quizzes
  • 🔗 host_id → users
  • 📌 pin (6 haneli)
  • 🔗 join_url (slug)
  • 📊 status (lobby/active/ended)
  • 👥 allow_anonymous
  • 🕐 started_at / ended_at

🏅 teams

  • 🔑 id
  • 🔗 game_session_id → game_sessions
  • 📝 name
  • 🎨 color

Opsiyonel. Host yoksa bireysel mod.

👥 players

  • 🔑 id
  • 🔗 game_session_id
  • 🔗 team_id (opsiyonel)
  • 🔗 user_id (opsiyonel)
  • 📝 nickname
  • 🕐 joined_at

💬 player_answers

  • 🔑 id
  • 🔗 player_id
  • 🔗 question_id
  • 🔗 option_id
  • ⏱️ answered_ms (kaç ms'de)
  • 🏆 points_earned

🏆 game_results

  • 🔑 id
  • 🔗 game_session_id
  • 🔗 player_id
  • 🔢 total_points
  • 🥇 rank
  • 📊 correct_count

PDF/CSV export için kalıcı kayıt.
